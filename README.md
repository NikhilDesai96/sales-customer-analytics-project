# Sales & Customer Analytics Project

## Project Overview
This project analyzes retail sales data using Python, SQL, and Power BI to generate business insights related to sales performance, profitability, customer behavior, and shipping efficiency.

---

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI

---

## Project Workflow
1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. SQL Business Analysis
4. Interactive Power BI Dashboard Creation

---

## Key Business Insights
- Technology category generated highest sales.
- West region contributed highest profit.
- Certain discounts negatively impacted profitability.
- Sales trends increased significantly during later months.
- Top customers contributed major portion of revenue.

---

## Dashboard Features
- KPI Cards
- Monthly Sales Trend
- Profit Contribution Analysis
- Top Customers Analysis
- Discount vs Profit Analysis
- Shipping Performance Analysis
- Interactive Filters
