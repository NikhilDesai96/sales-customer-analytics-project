# 📊 Sales & Customer Analytics Project

## 📌 Project Overview
This project analyzes retail sales data using **Python, SQL, and Power BI** to uncover business insights related to sales performance, profitability, customer behavior, discounts, and shipping efficiency.

The project demonstrates a complete end-to-end analytics workflow including:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- SQL Business Analysis
- Interactive Dashboard Development

# 🔍 Key Business Insights

- Technology category generated the highest sales.
- West region contributed the highest profit.
- High discounts negatively impacted profitability.
- Sales increased significantly during later months.
- Top customers contributed major revenue share.
- Shipping performance varied across regions.

# ⚙️ Project Workflow

## 1️⃣ Data Cleaning & Preprocessing
- Handled missing values
- Converted date columns
- Created calculated columns
- Prepared data for analysis

## 2️⃣ Exploratory Data Analysis (EDA)
- Sales analysis
- Profit analysis
- Customer analysis
- Category performance analysis

## 3️⃣ SQL Business Analysis
Performed SQL queries using:
- SELECT
- WHERE
- GROUP BY
- HAVING
- CASE WHEN
- Aggregate Functions

## 4️⃣ Power BI Dashboard
Built an interactive dashboard for:
- sales tracking
- profitability analysis
- customer insights
- operational analysis

# 🚀 How to Run the Project

1. Open the Jupyter Notebook from the `notebooks` folder.
2. Run SQL queries from the `sql` folder.
3. Open the Power BI dashboard from the `powerbi` folder.
4. Explore the dashboard using slicers and filters.
